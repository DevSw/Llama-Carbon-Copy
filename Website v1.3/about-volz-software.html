The Creation of Llama Carbon Copy

I wrote Llama Carbon Copy because it was useful for me in my consulting business. Often I have to travel to a customer's location, which necessitates bringing my laptop and all the code that I have for that customer on my desktop. I've setup Llama Carbon Copy to make it easier to move the information from my desktop to my laptop.

Some friends showed some interest, indicating that they might have more creative uses for the software (everything from standard document delivery in a business setting, to synchronizing their music). It seemed that it filled a very specific, well defined need. I set out to extend my internally used code so that it would be easy to install and use on other people's machines. The result is Llama Carbon Copy.
